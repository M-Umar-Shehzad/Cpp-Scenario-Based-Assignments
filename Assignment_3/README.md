# C++ Learning Management System

A simple command-line Learning Management System built in C++ for learning basic programming concepts.

## About

This program provides a menu-driven interface to explore fundamental C++ topics with a basic login system.

## What it consists of

- Login system with ID and password
- Menu with 4 C++ topics:
  - Introduction (Syntax, Variables, Data Types)
  - Conditional Statements (if, if-else, else-if)
  - Loops (For, While, Do-While)
  - Jump Statements (Break, Continue, Go-To)

## Concepts Used

- Functions
- While loops
- Switch statements
- Boolean variables
- Input/Output operations
